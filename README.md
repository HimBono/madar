# Madar — مدار
### Digital Agency Portfolio · GCC

A bilingual (EN/AR) portfolio and landing page for Madar Digital Agency.

## Stack
- Pure HTML / CSS / JS — zero dependencies
- Google Fonts (Barlow Condensed, DM Sans, Cairo)
- Deployed on Vercel

## Features
- Full RTL Arabic support with language toggle
- Animated orbit SVG hero
- Portfolio grid with category filtering
- Contact form
- Fully responsive

## Deployment
Deployed automatically via Vercel on every push to `main`.

## Adding Projects
In `index.html`, find the `#projects-grid` section and duplicate a `.project-card` block.  
Set `data-category` to one of: `marketing` · `dev` · `social` · `it`
